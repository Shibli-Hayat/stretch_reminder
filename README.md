# Stretch Reminder

A simple and lightweight Python utility that runs in the background or terminal, prompting you with a pop-up window every 30 minutes to stand up, stretch, and reduce screen fatigue.

## Features
- Minimalist Tkinter popup dialog.
- Built-in test mode (opens first popup within 5 seconds for instant verification).
- Repeats automatically every 30 minutes.
- Easily exit using `Ctrl+C` in the terminal.

## Requirements
- Python 3.x (uses built-in `tkinter` module, no extra packages required).

## How to Run
Run the script using:
```bash
python script.py
```
